## How to run
- Clone or fork this project on your local.
- Create file `.env`
```go
export DATABASE_USERNAME=<username-db>
export DATABASE_PASSWORD=<password-db>
export DATABASE_HOST=<host-db>
export DATABASE_NAME=<name-db>
```
- Run project
```go
$ npm start
$ npm run dev

- Install package
```go
$ install express
$ install dotenv
$ install sequelize
$ install mysql2
